# OPEN_SOURCE_NOTES

## Positioning
This should be presented as an early but working prototype for structured AI memory continuity.

## What to claim
Safe to claim:
- The prototype can ingest sessions
- It can retrieve accumulated context
- It can ingest durable update notes
- It demonstrates multi-session memory accumulation

Do not claim yet:
- production readiness
- enterprise reliability
- complete benchmark superiority
- full Codex/Claude integration
- complete conflict-resolution maturity

## Suggested first publish framing
An early prototype for a portable, persistent, mergeable AI memory layer.

## Important evidence
Use the artifacts folder as proof.

## Next public milestone
A benchmarked and cleaner workflow demo with Codex CLI and Claude Code.
