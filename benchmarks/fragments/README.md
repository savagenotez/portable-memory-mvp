# Fragments

This folder contains phrase-fragment attribution outputs.

## Purpose

These reports try to identify which fragments inside hybrid-added phrase-saver lines are carrying the restoring signal.

## Why this matters

A full line may be larger than necessary.
Sometimes only a fragment, or a title plus a fragment, is enough to preserve meaning.

## Current outputs

- JSON fragment attribution report
- Markdown fragment attribution report

- phrase-fragment-per-byte mode
