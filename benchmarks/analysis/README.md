# Analysis

This folder contains per-scenario benchmark analysis outputs.

## Purpose

These reports show which mode wins for each scenario under different criteria:

- best overall
- best above recall threshold
- safest
- cheapest

## Why this matters

This helps identify:
- when compression is enough
- when phrase-saver escalation is needed
- when hybrid fallback is required
- whether the classifier and threshold-gated controller differ on specific scenarios
