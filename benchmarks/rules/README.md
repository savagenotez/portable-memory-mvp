# Rules

This folder contains mined upgrade rules derived from diagnostics reports.

## Purpose

These rules try to convert diagnostics into actionable memory-policy guidance.

## Current outputs

- JSON rule file
- Markdown rule report

## Current rule themes

- keep durable core lines
- preserve phrase-saving situational lines
- collapse redundant lines
- compress costly lines unless they save recall

## Future direction

- learn rules across many runs
- compare winning and losing runs
- auto-apply rules inside benchmark modes
- derive adaptive memory policies
