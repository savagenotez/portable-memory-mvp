# Adversarial Validation

This folder contains adversarial validation outputs for pruning and context-preservation modes.

## Purpose

These reports test whether a mode still preserves meaning when:
- expected phrases are varied
- wording is softened or paraphrased
- exact substring matching is no longer the only signal

## Why this matters

A mode can look strong under exact phrase checks but fail when wording shifts.
This validation helps determine whether the preservation behavior is robust or only exact-match friendly.
