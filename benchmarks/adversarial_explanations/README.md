# Non-Pruning Winner Analysis

This folder contains analysis of the adversarial scenarios where phrase-saver pruning did not win.

## Purpose

These reports explain:
- which 2 scenarios were not won by phrase-saver pruning
- what mode beat it
- whether the reason was higher adversarial robustness, better overlap, or simply more retained context

## Why this matters

This helps identify the remaining failure cases for pruning and shows what must be preserved to make pruning even stronger.
